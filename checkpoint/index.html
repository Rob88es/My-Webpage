¿Cuáles son algunas de las cosas que hacen que SCSS sea diferente de CSS?

Nos da la facilidad de que incluir variables o mixin, los cuales se repiten constantemente por todo el proyecto, esto a mayor escala se hace una forma mas simple de escribir el codigo y organizarlo, para que otros programadores lo puedan entender. Ademas hace que el programa sea mas ligero y cargue mas rapido.

Un ejemplo seria usar una variable para los cambios de color de todas las etiquetas un los posts, header, sin tener que hacerlo uno por uno.

¿Qué es una variable SCSS? (porque crees que debes utilizarla pon un ejemplo de una variable, escribe una variable y como se pondría para utilizarla)

Una variable es un valor que se repite en varias partes del proyecto, podemos asignar una variable para cambiar un valor determinado en todas partes con una simple operacion, sin tener que pasar cambiando individualmente.

$color_oscuro: black;

.gridFour { color: $color_oscuro; }

¿Qué es un SCSS Mixin? (porque crees que debes utilizarla pon un ejemplo de un mixin, escribiendo cómo se crea y como se pondría para utilizarla)

El SCSS Mixin es una potente herramienta al momento de hacer el codigo mas corto y simple, ya que existen muchos varoles que se repiten de manera constante. Asi que es como tener una biblioteca de recursos que usamos frecuentemente en el proyecto

@mixin grid-container() { display:grid; width: 500px; background-color: lightgray; }

.gridFour { @include grid-container; }

¿Qué significa Unidad fraccionaria (fr) con CSS Grid?

La unidad de fraccionaria (fr) es la cantidad de filas y columnas que dividimos un espacio de trabajo como si se tratara de una tabla en excel, esto nos permite tener un control preciso de donde queremos trabajar en la tabla y asignarle ciertas reglas sobre su funcionamiento.
